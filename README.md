## DESCRIÇÃO
---
1.    O projeto deve ser implementado com MVC
2.    As frases do desafio estão no arquivo ‘frases.txt’ em que cada linha é uma frase (Arquivo)
3.    Cada frase deve ser lida do arquivo e armazenada em um ArrayList de objetos do tipo Frase (Arquivo, Generic e Exceção).
4.    A classe Frase deve ter somente uma string e um método booleano que compara a string que é atributo com uma outra string passada como parâmetro 
5.    A cada vez que o usuário clicar no botão ‘Confirmar’ ou ao apertar enter, a frase digitada será comparada a frase do desafio (Eventos de GUI). 
6.    Independentemente se a frase for correta ou errada, passa para a próxima frase do ArrayList 
7.    Antes de mostrar uma nova frase será necessário mostrar uma mensagem falando se a frase digitada foi correta ou errada.
8.    Ao finalizar o programa, um arquivo deve armazenar o score máximo da quantidade de frases corretas. 
9.    Quando todas as frases forem mostradas, o programa fecha e mostra o score do jogador e o score máximo que já aconteceu no jogo
